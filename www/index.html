Synopsis

Figure 1
MouseBot Chassis

The UNCo MouseBot is a small, four-wheeled robot originally designed by UNC alum Rob Shiely. The MouseBot is a derivative of the "Lazarus" project. Now, the MouseBot project has become a project that is being built upon by the UNC Physics Robotics class, advised by Dr. Semak.

This project is actively seeking interested students who want to gain experience in mechanical and electrical design, software development, project management, and documentation. Contact Dr. Matt Semak at matthew.semak@unco.edu

To make the project more accessible to new users, all existing hardware design, software, and documentation (including the source for this web page) have been compiled into a github repository. Visit the page on github (MouseBot on Github) to find out more.

You can clone your own copy of the to start using/contributing to the project right away! For example, on Linux you would issue the following command from the command line:

$ git clone http://github.com/mowoodsiii/unco-mousebot.git

Members

The project is currently headed by Jessica Trujillo and advised by Dr. Matthew Semak. Significant design contributions have been provided by RoadNarrows
